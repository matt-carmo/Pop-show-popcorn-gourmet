## Installation

Install my-project with npm


### steps

##### 1. clone repository
```bash
git clone https://github.com/matt-carmo/pop-show-popcorn-gourmet
cd pop-show-popcorn-gourmet

```
##### 2. Install dependencies:
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install

```
##### 3. Start the development server:
```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install

```
### 4. Open the project in your browser: 
- Go to [http://localhost:3000](http://localhost:3000) or your IPV4 Address
